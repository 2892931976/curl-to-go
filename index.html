curl-to-Go

Instantly convert curl commands to Go code

This tool turns a curl command into Go code. It currently handles the following options: -X/--request, -d/--data, -H/--header, and -u/--user. It also understands JSON content types. As always, be sure to go fmt your code and pay attention to imports. For a quick way to generate a Go struct from JSON, see JSON-to-Go. Also feel free to contribute on GitHub.
